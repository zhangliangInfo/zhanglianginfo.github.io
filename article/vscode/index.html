Vscode配置信息

Thu, Aug 13, 2020

一、 中文配置

  1. 实用快捷键Ctrl + Shift + P，打开命令窗口，输入Configure Display Language，选择zh-cn
  2. 这时按照提示点击restart重启编辑器，可以是界面并没有变成中文界面，这时因为vscode默认没有安装中文包
  3. 在扩展应用中搜索：chinese，点击安装，然后重启，就可以看到界面变成了中文界面

二、 git配置

vscode中源代码管理中若提示当前没有源代码管理程序进行注册，则有可能是git路径配置错误：

  1. 查看当前git path

    $ where git

  2. 点击文件> 首选项 > 设置或使用Ctrl + ,快捷键，搜索git.path, 点击"在settings.json"中编辑，用户设置处，添加或修改"git.path"的属性值，保存；

  3. 这时打开git项目之后，会自动关联，每次删除修改都会在源代码管理里面显示修改的内容

  4. 推荐GitLens插件，可以在代码中显示每一行的修改信息(author, date, comment)
